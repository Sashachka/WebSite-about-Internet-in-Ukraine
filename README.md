More about this:

First page called "Home" is a page with chortcuts to different pages with articles about history of Inter et in Ukraine. Nothing special, but has circles with pages, forth paragraphs which consist with shortcuts.

Second page called "Search' this page with help of some functions and algorithm searches word or letters user inputs to search bar, and after it find word/letter it shows all pages with a little bit of context(shows a little bit of the sentence it find it in) and makes shortcut to page it found it. Also it creates highlight to all word/letters user inputed(click search->input word->short cut to page->highlight all same words/letters).

Third page is "Articles" contains 5 distinct pages with different content, page 1 is for how internet appeared in Ukraine, second hoe it was developing and some thoughts from people, 3rd is also about developing but with more official and academic style text, 4th is for soeculations, predictions and maybe future of Ukranian and global internet. In case of webPage it's also gets input from profile page but with one limitation only once and then overwrittes it with new input from user.
